This section shows predictions from two independent machine learning models:
  • Random Forest (RF)
  • XGBoost (XGB)
Each model outputs a probability score between 0 and 1 indicating how likely the PDF is tampered. How to interpret:
  • Higher value → stronger indication of tampering
  • Lower value → stronger indication of authenticity
Why two models? We use multiple models to improve robustness and reduce bias from a single algorithm. Differences between RF and XGB help identify uncertain or borderline cases.
